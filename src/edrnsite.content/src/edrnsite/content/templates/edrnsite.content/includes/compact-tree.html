      • Datasets

        Datasets

        A dataset contains a set of data files but can also contain other datasets, forming a nested composition. Datasets containing files cannot contain other datasets.

        Attributes that are not specified for a dataset inherit their values from their parent datasets or parent collection.

        Attributes

        CDERequired
        DatasetName✓
        DatasetDescription✓
        • Core Files

          Core Files

          Core files are the actual computer files of data that comprise the scientific instrument output of observations, experiments, and so forth.

          Attributes that are not specified for a core file inherit their values from their parent datasets or parent collection.
