• LabCAS Common Data Elements

    LabCAS CDEs

    The LabCAS Common Data Elements are used throughout the Laboratory Catalog and Archive System.

    • Collections

      Collections

      A collection is the highest-level structural element for organizing data in LabCAS. Collections contain datasets.

      • Attributes

        Collection Attributes

        The attributes of a collection are used to describe high-level details of entire sets of data and files.

      • Contained Objects

        Objects Within Collections

        Collections contain datasets, which themselelves may contain datasets.

        • Datasets

          Datasets

          A dataset contains a set of data files but can also contain other datasets, forming a nested composition. Datasets containing files cannot contain other datasets.

          Attributes that are not specified for a dataset inherit their values from their parent datasets or parent collection.

          • Attributes

            Dataset Attributes

            The attributes of a dataset inherit values from their parent dataset or collection if not otherwise specified.

            • CDE: DatasetName

              DatasetName

              Required
              Yes
              Purpose
              A short name for the dataset.
              Data Type
              Free text
            • CDE: DatasetDescription

              DatasetDescription

              Required
              Yes
              Purpose
              A summary of the data captured by this dataset.
              Data Type
              Free text
          • Contained Objects

            Objects Within Datasets

            Datasets can contain other nested datasets. "Leaf" datasets may also contain Core Files, Pathology Imaging Extensions, BRSI Imaging Extensions, and DICOM Extensions.

            • Core Files

              Core Files

              Core files are the actual computer files of data that comprise the scientific instrument output of observations, experiments, and so forth. Core Files do not contain any child objects.

              Attributes that are not specified for a dataset inherit their values from their parent datasets or parent collection.

              • Attributes
